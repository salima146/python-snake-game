#Snake Game 🐍

##Overview

This is a classic Snake game built in Python using Pygame. The game focuses on smooth movement, collision detection, and score tracking.

##Features

-Smooth snake movement
-Food spawning system
-Collision detection (walls and self)
-Score tracking
-Simple game loop structure

##Technologies Used

-Python
-Pygame

##How to Run

Make sure Python is installed.
Install dependencies using: pip install pygame
Run the game using: python main.py
